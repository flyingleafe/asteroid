class asteroid.masknn.wavenet.DemucsNet(chin=1, chout=1, hidden=48, depth=5, kernel_size=8, stride=4, causal=True, resample=4, growth=2, max_hidden=10000, normalize=True, glu=True, rescale=0.1, floor=0.001)[source]¶

Bases: torch.nn.Module

Demucs speech enhancement model. :param - chin: number of input channels. :type - chin: int :param - chout: number of output channels. :type - chout: int :param - hidden: number of initial hidden channels. :type - hidden: int :param - depth: number of layers. :type - depth: int :param - kernel_size: kernel size for each layer. :type - kernel_size: int :param - stride: stride for each layer. :type - stride: int :param - causal: if false, uses BiLSTM instead of LSTM. :type - causal: bool :param - resample: amount of resampling to apply to the input/output.

Can be one of 1, 2 or 4.

Parameters

  • growth (-) – number of channels is multiplied by this for every layer.

  • max_hidden (-) – maximum number of channels. Can be useful to control the size/speed of the model.

  • normalize (-) – if true, normalize the input.

  • glu (-) – if true uses GLU instead of ReLU in 1x1 convolutions.

  • rescale (-) – controls custom weight initialization. See https://arxiv.org/abs/1911.13254.

  • floor (-) – stability flooring when normalizing.

valid_length(length)[source]¶

Return the nearest valid length to use with the model so that there is no time steps left over in a convolutions, e.g. for all layers, size of the input - kernel_size % stride = 0.

If the mixture has a valid length, the estimated sources will have exactly the same length.
